# iTerm Custom Theme

![iTerm2](https://github.com/hackerspider1/dotfiles/blob/main/Terminal.jpg)

## Install Brew
`
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
`
## Install iTerm2
`
brew install --cask iterm2
`
## Install Starfish
`
brew install starship
`
## Install Fig
`
brew install --cask fig
`

## Install exa
`
brew install exa
`

Once everything is installed copy the .zshrc file from https://github.com/hackerspider1/dotfiles/main/.zshrc and starfish config file from https://github.com/hackerspider1/dotfiles/main/.config/starship.toml into `.config/` folder.
